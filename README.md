> It was at this moment that he knew...

Play famous "He f*cked up" sound.

## Compatibility

- Linux
- Windows
- Mac

## Installing
Install globally:

    npm install -g hef_ckedup

## Running
Run command:

    hef_ckedup

Or run via npx command:

    npx hef_ckedup


Or import in NodeJS script:

    import hefckedup from 'hef_ckedup';

    hefckedup();
